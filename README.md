# WiFiCarClient
本人2014年本科毕业设计，基于OpenWrt的视频监控WiFi遥控车的Android客户端源码

主要功能：
1. 通过socket连接小车上的ser2net进而向串口发送信息控制小车运动方向的云台的代码
2. 从小车上获取mjpeg视频流并截图，录像。
